SuperBlock

OS managed structure used to describe the number of disk blocks, the number of inodes, and the block number of the head block of the free list.

public int totalBlocks

The number of disk blocks available to this file system, including this superblock.

public int totalInodes

The number of Inodes available to this file system.

public int freeList

The blockID of the free listâ€™s head.

public SuperBlock(int diskSize)

Create a new SuperBlock with the given disk size. Disk size must be greater than 0.

public void sync()

Write totalBlocks, totalInodes, and freeList to disk superblock.
Returns 0 on success, -1 on failure.

public int getFreeBlocks()

Dequeue the top block from the free list, increment freeList to the next free block.
Returns dequeued blockID on success, -1 on failure.

public void returnBlock(int blockID)

Enqueue the given block to the end of the free list.
Returns 0 on success, -1 on failure.
