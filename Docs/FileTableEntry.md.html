FileTableEntry (FTE)

Record a file used by the parent thread.

public int seekPtr

The file seek pointer. Records the position of this FTE within the file in bytes.

public final Inode inode

The Inode this FTE has registered.

public final short iNumber

This Inode number.

public int count

The number of threads sharing this entry.

public final String mode

The access mode of this FTE. See FileSystem for more information.

public FileTableEntry(Inode inode, short inodeNumber, String mode)

Construct a FTE with the given Inode, inodeNumber, and mode. Sets count = 1, and places seekPtr based on the given mode.
